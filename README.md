# 使用技術/開発環境

### 使用PC
- macOS Catalina 10.15.7

### フロントエンド
- JavaScript
- Nuxt.js
  - @nuxt/cli v2.14.7
  - npm 6.9.0
  - node v10.16.0
 
### サーバーサイド
- firebase

  - fireAuth
- TwitterAPI

### デプロイ
- fire Hosting
